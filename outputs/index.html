Why recall, not accuracy

The cohort is ~19% positive — realistic for a rare disease. A model that always predicts healthy would score 81% accuracy while catching nobody. Because a missed Brugada case carries real risk of sudden cardiac death, this project reports recall on the Brugada class as the headline metric, and tunes the decision threshold toward catching cases (Triage tab).

Spectral basis: averaged across the cohort, Brugada patients show roughly 2× lower high-frequency (15–40 Hz) power in V1 (0.099 vs 0.183 relative power) — a consistent, learnable difference, consistent with prior spectral-ECG findings in Brugada syndrome (García-Iglesias et al., 2019).

Pipeline & honest limitations

Pipeline: WFDB ingest → Butterworth band-pass (0.5–40 Hz) → FFT + morphology features (V1–V3) → SQLite cohort analysis → class-weighted classifier (Logistic Regression & Random Forest, stratified 5-fold CV) → this dashboard.

Limitations: single-center cohort, 12-second recordings, only 69 positive cases. This is a proof-of-concept screening aid, not a diagnostic. A deep model was deliberately avoided — the data volume doesn't support it, and a regularised classical model is more honest and defensible here.
